🔹 핵심 요약
부모 클래스 생성자
super() 없이 자기 필드만 초기화 (Alt + Shift + S → O 로 생성 가능)
자식 클래스 생성자
부모 필드(name, age)를 초기화하려면 super(값, 값) 호출 필수
super() 호출 뒤에 자기 필드 초기화
Alt + Shift + S → O 로 생성 후 직접 super(...) 추가해야 함
자식 클래스에서 super()를 안 쓰면 오류 발생
부모 필드를 초기화할 방법이 없기 때문! (컴파일러가 기본 생성자를 찾지 못함)
✅ 정리하면?

부모 클래스는 자기 생성자를 직접 만들고 (Alt + Shift + S → O)
자식 클래스는 super(...)를 이용해 부모의 생성자를 호출해야 한다! 🚀

🔹 toString() 자동 생성
단축키
👉 Alt + Shift + S → S (Generate toString())

의미 및 설명
toString() 메서드는 객체를 출력할 때 자동으로 호출되며, 객체의 정보를 문자열 형태로 반환하는 역할을 합니다.
보통 객체의 필드 값들을 출력하는데 사용됩니다.

🔹 Getter & Setter 자동 생성
단축키
👉 Alt + Shift + S → R (Generate Getters and Setters)

의미 및 설명
Getter(접근자): 클래스의 private 변수 값을 가져오는 메서드
Setter(설정자): 클래스의 private 변수 값을 변경하는 메서드
이클립스에서 자동 생성하면, 변수에 대한 getter와 setter가 자동으로 만들어집니다.
