리눅스 주요 사항 정리
=====================================
- 리눅스 주요 디렉토리
 / : 루트 디렉토리. 최상위 폴더 
 /bin : 명령어 바이너리(기본 유틸, 예: ls, cp) 폴더
 /boot : 부팅 관련 폴더
 /dev : 장치 파일 폴더
 /etc : 설정 파일 폴더, 환경설정 파일 위치, 대부분 파일들은 텍스트파일
   --            /etc/passwd : 유저의 각종 정보 저장
                 /etc/shadow : 유저의 패스워드를 암호화하여 저장하는 파일

 /home : 개별 사용자의 홈 폴더가 있는 곳
 /lib : 필수 라이브러리
 /mnt : 마운트된 미디어 폴더
 /proc: 시스템 정보를 보여주는 특수 파일시스템
 /root : root 사용자의 홈 폴더
 /sbin : 필수 시스템 바이너리(주로 시스템 관리자들을 위한 프로그램 위치)
 /tmp : 임시 파일 폴더
 /var : 자주 변경되는 파일 폴더(주로 로그 파일)
